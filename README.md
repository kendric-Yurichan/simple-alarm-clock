# Simple Alarm Clock

A minimal, single-file alarm clock that runs entirely in the browser. No server, no build tools, no dependencies.

## Features

- Live digital clock with seconds and date
- Add alarms with an optional label
- Enable / disable alarms without deleting them
- Audible chirp sound via Web Audio API (no sound files needed)
- Browser desktop notifications (optional, prompted on first add)
- Alarms persist across page reloads via localStorage

## Usage

Just open `index.html` in any modern browser. That's it.

## Compatibility

Chrome 66+, Firefox 76+, Safari 14+, Edge 79+
